# 💪 BMI Calculator

A simple and interactive BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript. This app calculates BMI based on user input and provides instant health category classification with a modern responsive UI.

---

## 🚀 Features

- ⚖️ Calculate BMI using weight (kg) and height (cm)  
- 📊 Instant BMI result display  
- 🏷️ Category classification:
  - Underweight 🥗  
  - Normal ✅  
  - Overweight ⚠️  
  - Obese 🚨  
- 🎨 Modern glassmorphism UI  
- 📱 Fully responsive design  
- ⚡ Fast and lightweight (no external libraries)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)

## ⚙️ How to Run

### 2️⃣ Navigate into the folder
cd bmi-calculator

### 3️⃣ Open the project

Double click:
bmi calculator.html

OR use command:

Windows:
start "bmi calculator.html"

Mac:
open "bmi calculator.html"

Linux:
xdg-open "bmi calculator.html"

## 🎯 Usage

1. Enter your weight (kg)  
2. Enter your height (cm)  
3. Click "Calculate"  
4. View:
   - BMI value  
   - Health category
